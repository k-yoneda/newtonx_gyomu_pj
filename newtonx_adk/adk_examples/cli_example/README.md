# NewtonX CLI Example

NewtonX ADKを使用したCLIアプリケーションの実装例です。

## 機能

- **タブキー補完**: ファイルパス入力時にタブキーで自動補完
- **ファイルアップロード**: 画像・ドキュメントファイルのアップロード
- **チャット機能**: リアルタイムチャット
- **アシスタント管理**: アシスタント一覧表示・選択
- **履歴機能**: コマンド履歴の保存・参照

## インストール

```bash
cd cli_example
pip install -r requirements.txt
```

## 使用方法

```bash
python cli_example.py
```

### 認証（公開クライアント / MSAL, PKCE優先）
- 事前に以下を実行してEntra ID情報を設定します:
  ```bash
  newtonx-config configure
  # または確実な実行（モジュール経由）
  python -m newtonx_adk.config_cli configure
  ```
- 実行時はサイレント→PKCE（ブラウザ）→デバイスコードの順で認証します。
- クライアントシークレットは不要です。

## 特徴

### タブキー補完
- ファイルパス入力時にタブキーで自動補完
- アップロード可能なファイル形式を優先表示
- ディレクトリとファイルの両方を補完

### ファイルアップロード
- 画像ファイル: jpg, jpeg, png, gif, bmp, webp
- ドキュメントファイル: pdf, docx, txt, pptx, xls, xlsx, md
- ファイルサイズ制限: 10MB以下

### 音声付きメッセージ送信
- 音声ファイル: wav, mp3, aiff, aac, ogg, flac
- ファイルサイズ制限: 15MB以下
- メインメニューから「音声付きメッセージ送信」を選択して使用できます
- 音声ファイルを添付してメッセージを送信すると、AIアシスタントが音声の内容を理解して応答します

### ユーザー体験
- 利用可能なファイル一覧表示
- 分かりやすいヒント表示
- エラーハンドリング
- コマンド履歴保存

## 追加機能: アシスタントへのナレッジ登録/削除

- メインメニューに以下を追加しました:
  - 「アシスタントへのナレッジ登録」: `.docx/.xls/.xlsx/.pptx/.pdf/.txt`（16MB以下）をアップロード
  - 「アシスタントナレッジ削除」: `file_uid` を指定して削除

## 追加機能: 音声付きメッセージ送信

- メインメニューに「音声付きメッセージ送信」を追加しました
- 対応形式: `.wav`, `.mp3`, `.aiff`, `.aac`, `.ogg`, `.flac`
- 最大ファイルサイズ: 15MB
- 音声ファイルを添付してメッセージを送信すると、AIアシスタントが音声の内容を理解して応答します

### 使い方

```bash
python adk_examples/cli_example/cli_example.py
```

1. メインメニューから「5. 音声付きメッセージ送信」を選択
2. チャットを選択
3. メッセージを入力
4. ウェブ検索/ナレッジ検索の設定を選択
5. 音声ファイルパスを入力（タブキーで補完可能）
6. 音声ファイルの検証後、メッセージが送信されます
