NewtonX Auto Runner サンプルドキュメント

このドキュメントは、NewtonX Auto Runnerのテスト用に作成されました。

## 内容

- 設定ファイルベースの自動実行
- YAML設定ファイルによるタスク定義
- 複数のタスクの連続実行
- 結果の自動保存

## 機能

1. **初期化**: プロジェクトの初期化
2. **ファイルアップロード**: サンプルファイルのアップロード
3. **チャット作成**: 新しいチャットの作成
4. **メッセージ送信**: テストメッセージの送信
5. **ファイル分析**: アップロードしたファイルの分析

## 設定

設定は `config.yml` ファイルで管理されます：

```yaml
project:
  name: "NewtonX Auto Runner Example"
  description: "設定ファイルベースの自動実行サンプル"
  version: "1.0.0"
```

## 使用方法

```bash
python auto_runner.py --config config.yml
```

このサンプルは、NewtonX ADKを使用した自動化ワークフローの実装例です。
